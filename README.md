# BountyMatch Copilot

Developer-first starter for a CLI + lightweight dashboard that helps contributors discover and evaluate paid OSS bounties.

## Quickstart

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .
bountymatch
```

Run tests:

```bash
pytest -q
```
